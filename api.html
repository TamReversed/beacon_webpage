API Reference

REST API for incidents, components, and subscribers. Use your API key in the Authorization: Bearer header.

Base URL

GET https://api.beacon.example/v1

Incidents

Create and manage incidents. All times are UTC, ISO 8601.

POST /incidents
GET /incidents
PATCH /incidents/:id
POST /incidents/:id/updates

Components

List and update component status. Status values: operational, degraded_performance, partial_outage, major_outage.

GET /components
PATCH /components/:id

Rate limits

100 requests per minute per API key. Headers: X-RateLimit-Limit, X-RateLimit-Remaining, X-RateLimit-Reset.
